# VS Code database
### Extension for Visual Studio Code 

## Usage
![how use](https://github.com/Bajdzis/vscode-database/raw/master/readme/v0.1.gif)

## Install
* Press `Shift+Ctrl+P` 
* Pick `Extensions: Install Extension`
* Find `vscode-database`
